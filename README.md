## 소개
* 주식회사 두베에서 소프트웨어 개발자를 채용합니다. 
* 트렌디한 개발을 꿈꾸시는 분들의 많은 참여 바랍니다.

### 채용절차
* 서류 심사 > 부서장 면접 > 최종 면접
* 경력증명서및 포트폴리오는 자유형식으로 작성후 채용담당자(thlee@do-best.co.kr / nixstory@gmail.com)에게 제출

### 업무 환경 및 복지
* 오전 9시 00분 출근 ~ 오후 6시 00분 퇴근
* 15인치+ 맥북 프로 제공
* 각종 세미나 및 컨퍼런스 참여 지원
* 재택근무 가능 

### 기술 스택
